# References

This directory contains external dependencies and reference implementations used by the project.

## Components

- iroh/
  - The core Iroh P2P networking stack.
  - Provides the foundation for peer-to-peer communication.

- iroh-gossip/
  - Use for PubSub messaging.
  - Enables efficient message propagation across the network.

- iroh-h3/
  - HTTP3 implementation over Iroh.
  - Facilitates HTTP-like communication over the P2P network.
